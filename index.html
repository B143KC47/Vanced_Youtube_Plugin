Quick Start · 快速开始

  1. Download the ZIP above and extract it / 下载上方 ZIP 并解压。
  2. Open chrome://extensions/, enable Developer mode / 打开扩展开发者模式。
  3. Click Load unpacked & choose the folder / 选择解压后的文件夹。
  4. Pin 🎯 icon & enjoy! / 固定 📌 图标，开始享受！
